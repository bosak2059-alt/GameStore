## 📖 О проекте

**GameStore** — это полнофункциональное одностраничное приложение (SPA) для покупки видеоигр, разработанное как учебный проект 3-го курса программирования. Проект демонстрирует глубокое понимание современных веб-технологий, паттернов проектирования и лучших практик фронтенд-разработки.

### 🎯 Цели проекта

- **Демонстрация навыков**: Показать владение чистым JavaScript без зависимостей от фреймворков
- **Современные практики**: Использовать ES6+ модули, CSS-переменные, View Transition API
- **Архитектура**: Реализовать компонентный подход, управление состоянием, роутинг
- **UX/UI**: Создать красивый, анимированный и доступный интерфейс
- **Мультиязычность**: Поддержка 7 языков и 4 валют
- **Реалистичность**: Имитация реального магазина с авторизацией, кошельком, историей покупок

### 🌟 Ключевые особенности

- ✅ **Чистый JavaScript** — без React, Vue или Angular
- ✅ **Компонентная архитектура** — модульный и переиспользуемый код
- ✅ **Реактивное состояние** — паттерн Observer для управления данными
- ✅ **SPA-роутинг** — навигация без перезагрузки страницы
- ✅ **Анимации** — 40+ CSS-анимаций и переходов
- ✅ **Мультиязычность** — 7 языков (RU, EN, JA, HY, ES, DE, FR)
- ✅ **Мультивалютность** — 4 валюты (RUB, USD, EUR, KZT)
- ✅ **Адаптивный дизайн** — работает на всех устройствах
- ✅ **Темная/светлая тема** — с плавными переходами
- ✅ **Авторизация** — регистрация, вход, сессии
- ✅ **Кошелёк** — внутренний баланс магазина
- ✅ **История покупок** — полный учёт транзакций

## 🛠 Стек технологий

### Основные технологии

| Технология | Версия | Назначение |
|-----------|--------|-----------|
| **JavaScript** | ES2022 | Основной язык программирования |
| **HTML5** | 5.3 | Разметка и семантика |
| **CSS3** | 3.0 | Стилизация и анимации |
| **Vite** | 5.0+ | Сборщик и dev-сервер |

### Архитектурные паттерны

- **Observer Pattern** — реактивное управление состоянием
- **Module Pattern** — инкапсуляция и организация кода
- **Component Pattern** — переиспользуемые UI-компоненты
- **Singleton Pattern** — глобальное хранилище состояния
- **Factory Pattern** — создание компонентов роутером
- **MVC (упрощённый)** — разделение данных, логики и представления

### Веб-стандарты и API

- **ES6+ Modules** — импорт/экспорт модулей
- **History API** — SPA-роутинг без хешей
- **LocalStorage API** — персистентность данных
- **Fetch API** — загрузка данных
- **Intl API** — форматирование валют и дат
- **View Transition API** — плавная смена темы (Chrome 111+)
- **CSS Custom Properties** — темизация
- **CSS Animations** — 40+ анимаций
- **CSS Grid & Flexbox** — адаптивная вёрстка

### Инструменты разработки

- **VS Code** — редактор кода
- **Git** — система контроля версий
- **npm** — менеджер пакетов
- **Chrome DevTools** — отладка

##  Функционал

###  Главная страница

- **Каталог игр** — 18 игр различных жанров
- **Поиск** — мгновенный поиск с debounce (300мс)
- **Фильтрация** — по категориям (RPG, Strategy, Action и др.)
- **Карточки игр** — с изображениями, ценами, скидками
- **Анимации** — stagger-появление карточек, hover-эффекты

### 🎮 Страница игры

- **Детальная информация** — описание, разработчик, рейтинг
- **Галерея** — изображение игры
- **Теги** — жанры и категории
- **Цена** — с отображением скидки
- **Добавление в корзину** — с toast-уведомлением

### 🛒 Корзина

- **Список товаров** — с изображениями и ценами
- **Управление** — добавление, удаление, очистка
- **Подсчёт** — автоматический расчёт суммы
- **Способы оплаты**:
  - 💰 С баланса магазина
  - 💳 Банковская карта
- **Оформление заказа** — с сохранением в историю
- **Empty state** — красивая заглушка для пустой корзины

### 👤 Авторизация

- **Регистрация** — с валидацией полей
- **Вход** — с проверкой учётных данных
- **Валидация**:
  - Email (формат)
  - Пароль (минимум 6 символов)
  - Имя пользователя (минимум 3 символа)
  - Подтверждение пароля
- **Индикатор силы пароля** — 5 уровней
- **Показ/скрытие пароля** — кнопка-глаз
- **Анимации** — плавающие круги, пульсация логотипа
- **Социальные кнопки** — Google, Steam, Discord (заглушки)

### 💰 Кошелёк

- **Баланс** — внутренний счёт пользователя
- **Пополнение** — через страницу `/wallet-topup`
- **Оплата** — списание с баланса при покупке
- **История транзакций** — все операции в профиле
- **Валидация** — проверка достаточности средств

### 📊 Профиль пользователя

- **Аватар** — с инициалом и анимацией
- **Информация** — имя, email, дата регистрации
- **Статистика**:
  - 💰 Текущий баланс
  - 🛍️ Количество покупок
  - 💸 Общая сумма потраченного
- **История покупок** — с деталями и датами
- **Управление** — кнопка выхода

## 🚀 Установка

### Системные требования

- **Node.js** 18.0 или выше
- **npm** 9.0 или выше
- **Браузер** с поддержкой ES6+ модулей (Chrome 90+, Firefox 88+, Safari 14+)

### Шаги установки

1. **Клонируйте репозиторий**

```
git clone https://github.com/yourusername/gamestore.git
cd gamestore
```

2. **Установите зависимости**
```
npm install
```

3. **Запустите dev-сервер**
```
npm run dev
```
